# Internal source material

This folder holds product strategy and planning documents that back the marketing site but are not part of the public site experience.

- `01_App_Blueprint.md` — high-level app blueprint
- `02_Product_Strategy.md` — positioning, audience, pricing
- `03_Feature_Inventory.md` — features by surface
- `04_Functional_Scope.md` — functional spec / behavior
- `05_Customer_Journeys.md` — primary user journeys

These files remain in the public repository but are excluded from search indexing via `Disallow: /internal/` in `/robots.txt` and are not linked from any page or sitemap. Don't link to them from public HTML.
